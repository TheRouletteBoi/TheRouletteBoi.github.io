About

Red Dead Redemption Hook allows you to use script mods (.sprx) on the PS3.

SDK

The SDK allows developers to make their own scripts. A simple example is included.

Donate

Want to donate? Click the button below!

Requirements

  • EBOOT (BLES01294)

    • Supported versions

  • 1.01 Red Dead Redemption: Game of the Year Edition (BLES01294)
  • any other game versions also work
